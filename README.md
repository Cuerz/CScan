## 🎸 Intro介绍
通过python爬虫，开发的GUI界面的扫描工具。支持通过`域名/IP`进行扫描

- 支持自己导入字典查询
- 支持多线程扫描
- 支持多个状态码扫描
- 支持超时丢弃当前站点

---
## 💡 使用方式
- 下载gui程序
- 在resources文件夹下添加需要使用的字典
- 输入扫描的域名（http://www.example.com/）
- 开始扫描
